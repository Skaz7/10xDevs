Building embedded projects.
Use will create an account by giving his username, email and password.
Then he will be able to log into his account and create a database of electronic parts and devices he owns, and describe to AI agent what he wants to build.
Agent proposes a project first based on parts owned by user, and if user doesn't own all necessary parts, Agent proposes a project, for which user has to buy least parts/devices.
Agent can search the web and give user hint where he can buy missing parts.
Service would prevent user to add parts and prompt projects other than electronic, so he wouldn't be able to add chicken and ask for fried rice recipe.
